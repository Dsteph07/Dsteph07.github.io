Movie Recommendation System with RAG

Python · RAG · LLM API

Retrieval-Augmented Generation based system that recommends movies from natural language descriptions. The user writes what kind of movie they want to watch (for example: “I want a movie that makes me laugh but might make me cry at the end”), and the system retrieves the most relevant movies using semantic embeddings, then asks a remote LLM (via API) to select and justify a ranked list of recommendations.

End-to-end pipeline: movie dataset preparation, embedding generation, similarity search, and orchestration of LLM calls for explanation and ranking.
